# black

Ever wanted a browser tab/window that's just the color you need?\
 https://sanghoon5499.github.io/Black-Screen/
 
How to use:

1. Choosing a Color
Select from these colors:\
|   asd  |     sd    |  ad    |
| ------ | --------- | ------ |
| red    | orange    | yellow |
| lime   | limegreen | green  |
| sky    | skyblue   | blue   |
| violet | purple    | black  |
| brown  | gray      | grey   |
| white  | pink      |        |

\\
Or, you can use any 3 or 6 digit hex-value
\

2. Send it!\
Hit enter once you've typed out what you want and the screen should update.
