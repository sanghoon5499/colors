# Colors

Ever wanted a browser tab/window that's just the color you need?\
 https://sanghoon5499.github.io/colors/
 
How to use:

1. Choosing a Color
- You can type any 3 or 6 digit hex-value (both lower and uppercase work)
- Or, select (type) from these colors:

|        |           |        |
| ------ | --------- | ------ |
| red    | orange    | yellow |
| lime   | limegreen | green  |
| sky    | skyblue   | blue   |
| violet | purple    | black  |
| brown  | gray      | grey   |
| white  | pink      |        |
|        |           |        |

2. Send it!
- Hit enter once you've typed out what you want and the screen should update.
